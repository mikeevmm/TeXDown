# LaTeXDown does:

+ Text formatting:
    - Bold
    - Emphasis/itallics
    - Underline
    - Strikeout
+ Multiple formulas
    - Numbered
    - Unumbered
    - Easy centered formulas
+ Inline code
+ block code
+ Author/Date/Title tags
+ Easy macro tags
+ Easy include tags
+ Easy
    - theorems
    - lemmas
    - corollaries
        - Named
        - Unnamed
+ Sections
+ Blockquotes
+ Lists
    - Ordered
    - Unordered
+ Tables
    - Ugly
    - Pretty
        - With captions
+ Pure LaTeX

# LaTeXDown doesn't yet do:

+ Links
+ Images
